2 Types of Bootstrap Containers: fixed and fluid

These are built-in CSS classes in Bootstrap.

Fixed Container - example:

Responds to media queries (ex:

@media (min-width: 992px)
.container {
width: 750px;
}

The fixed width changes at certain breakpoints.

Fluid Container - example:

Only responds to viewport width.

Is 100% the width of its parent element.

Use fluid containers if you're doing a grid.
